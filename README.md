# Number Guessing Game

Hey! 
This is a simple Number Guessing Game I built using Python and Streamlit.

The idea is easy — the app generates a random number between 1 and 100, and you have to guess it. It will guide you by telling if your guess is too high or too low until you get it right.

---

## What this project does

* Lets you guess a number between 1 to 100
* Gives hints like "Too High" or "Too Low"
* Shows a success message when you guess correctly
* Simple and clean interface using Streamlit

---

##  Built with

* Python
* Streamlit

---

##  How to run this project

1. Install the required library:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

---

## Why I made this

I created this project to practice Python basics and learn how to build simple web apps using Streamlit. It’s a beginner-friendly project but helped me understand user input, logic building, and UI.
